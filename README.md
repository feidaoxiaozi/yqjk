    1.项目描述
    本项目是完成一个舆情监控系统，系统主要由六个菜单按钮，包括新增任务、启动采集、监测报告、转载列表以及用户注册和修改密码。
    点击新增任务，可以添加一个新的任务，添加成功后即点击启动采集，系统开始在各大新闻网站进行抓取新闻信息，采集完成后，转到系统主页面，显示跟任务相关的新闻列表，列表里有两个比较重要的链接，最新报告和转载列表，点击最新报告可以查看某任务相关的信息，包括点击量、转载量、类型、来源、URL等，点击转载列表，可以查看所有任务相关的新闻文件。
    另外，本系统需要对用户进行管理，用户类型有管理员和普通用户，普通用户需要注册登录到系统进行使用，管理员指定账号和密码，以保证整个系统的安全性。

    2.环境搭建
    本系统所用的开发环境有MyEclipse6.5+jdk1.6+tomcat6.0+mySql5.0.
    3.数据库设计
    本项目涉及到3张数据表，分别是yqjk_report.sql，yqjk_task.sql和yqjk_user.sql，其中前两张表有主外键关联关系，yqjk_report表中的主键reportId作为yqjk_task表的外键，以保证每次点击一个任务时都能查看到与之相关的新闻统计信息。
